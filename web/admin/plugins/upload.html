Plugin Upload Guidelines

  • Only .so (shared object) files are supported
  • Maximum file size is 50MB
  • Plugins must be compiled with the same Go version as the server
  • Ensure your plugin exports a "NewPlugin" function
  • Test plugins in a development environment first
